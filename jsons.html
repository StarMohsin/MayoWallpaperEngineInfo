Json Files

Mayo Wallpaper Engine supports playlists using json file, a json file can contain separate arrays of file paths for image/video and audio files. this means that a json file can be configured to create a playlist, that can contain a single video/image and multiple audio files and vice versa. The MayoPlayer can play a list of video or images along with some audio files, contructed by Mayo User Iterface in .json/.mss/.mpl formats.
MayoPlayer supports playlist
MayoPlayer supports slideshows
